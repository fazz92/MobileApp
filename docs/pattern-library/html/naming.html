Naming Conventions

.element

A root selector that decides the base styles of an element. This selector and its rules must be at the root of the less file, and at the top (cascading matters)

HTML

<a class="element">Book A Room</a>

LESS

.element {
    //base styles
}

.element-modifier

A modifier changes the dimensions and details of an element. Its name includes the selector it is extending, and a dash ('-')

HTML

<a class="element-large" href="#">Book</a>
<a class="element-medium" href="#">A</a>
<a class="element-small" href="#">B</a>

LESS

.element-large:extend(.btn) {
    //modifying styles
}
.element-medium:extend(.btn) {
    //modifying styles
}
.element-small:extend(.btn) {
    //modifying styles
}

.element style

A style changes the aesthetics of an element, using a semantic selector

HTML

<a class="element-large cta" href="#">Book</a>
<a class="element-medium cta" href="#">H</a>
<a class="element-medium alert" href="#">S</a>

LESS

.cta {
    background:@brand-gold-light;
    color:white;
}
.btn-medium.alert {
    color:white;
    background:red;
    border-color:white;
}
